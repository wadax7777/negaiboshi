# Ver10 Ultimate 開発ロードマップ

## 第1段階：土台の安定化
- 現在の地球・日本・お住まい・月・ISS表示を端末別に確認
- スマートフォンで地図が表示されない条件を特定
- 画像・API・Three.js読込エラーを整理
- キャッシュ対策とバージョン番号を統一

## 第2段階：地球のリアル化
- 地球表面テクスチャの高精細化
- 雲レイヤーを別球体としてゆっくり回転
- 太陽方向から昼夜境界を計算
- 夜側だけ都市光を表示
- 大気の薄い青い縁を追加

## 第3段階：宇宙空間のリアル化
- 星を単なる点ではなく距離感のある複数層で表示
- 天の川の淡い帯を背景へ追加
- カメラ移動に慣性と減速を追加
- 光り過ぎないブルームとレンズフレアを調整

## 第4段階：月・宇宙機
- 月の8Kテクスチャと高さ情報を最適化
- ISS、こうのとり、ハッブル、JWSTの表示
- 宇宙機を選ぶと説明と現在位置を表示
- 地球→ISS→月の自動航行モード

## 第5段階：物語と手紙
- 「ISSから手紙が届きました」の演出
- こうのとりが幸せを運ぶシーン
- 星座を選ぶと神話・童話を表示
- 流れ星に願いを書ける機能

## 第6段階：完成調整
- PC、Android、iPhoneで速度を確認
- 低性能端末向け軽量モード
- 音あり／音なし切替
- 読み込み中の美しい演出
- 公開前バックアップと復元確認
