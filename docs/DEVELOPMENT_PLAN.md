# Ver10 Ultimate 実装計画

## 1. リアル地球
Three.jsで地球本体、雲、大気、都市光を別レイヤーに分けます。雲だけを少し速く回転させ、夜側の都市光は光源方向に応じて表示量を変えます。

## 2. 昼夜境界
現在時刻から太陽方向を求め、地球シェーダー内で昼面と夜面を滑らかに混ぜます。境目を真っ二つにせず、夕暮れの薄い帯を設けます。

## 3. カメラ演出
地球全体、日本、お住まい、ISS、月を単純な切替ではなく、滑らかな補間でつなぎます。急な回転を避け、目的地へ近づくほどゆっくり減速させます。

## 4. 端末性能対策
PCでは高精細、スマートフォンでは画像解像度・星数・光表現を自動調整します。低速回線では先に軽量画面を表示し、後から高精細データへ切り替えます。

## 5. 物語演出
リアルな宇宙の中へ、手紙、願い星、こうのとりを控えめに追加します。常時派手に動かさず、利用者が選んだ時だけ始まる演出にします。

## 6. 推奨ファイル分割
- js/earth-real.js：地球・雲・都市光・大気
- js/space-camera.js：カメラ移動
- js/spacecraft.js：ISS・こうのとり等
- js/story-letter.js：手紙と願い星
- css/space-effects.css：画面演出

## 7. 実装順序
まず「地球全体」のリアル化を完成させ、その後に日本、お住まい、ISS、月へ広げます。一度に全部を変更せず、各段階でスマートフォン表示を確認します。
